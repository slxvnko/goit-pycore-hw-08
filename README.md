# AddressBook Assistant Bot

## Опис

Це консольний бот для керування адресною книгою. Підтримує:

- Додавання та зміна контактів і телефонів
- Додавання та перегляд днів народження
- Показ усіх контактів
- Показ контактів, які мають дні народження наступного тижня
- Збереження даних між сеансами за допомогою pickle

## Встановлення

1. Клонувати репозиторій або зберегти `bot.py`.
2. Запустити командою:
```bash
python bot.py
Команди

add [ім'я] [телефон] — додати новий контакт або телефон

change [ім'я] [старий телефон] [новий телефон] — змінити телефон

phone [ім'я] — показати телефони контакту

all — показати всі контакти

add-birthday [ім'я] [дата DD.MM.YYYY] — додати день народження

show-birthday [ім'я] — показати день народження контакту

birthdays — показати дні народження на наступний тиждень

hello — вітання від бота

close або exit — закрити програму

Формат даних

Телефон: 10 цифр

День народження: DD.MM.YYYY

Збереження даних

Дані автоматично зберігаються у файлі addressbook.pkl при закритті програми і завантажуються при запуску.
